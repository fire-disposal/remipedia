## 📋 推荐的 Clippy 配置

创建 `.clippy.toml`:

```toml
# 允许非关键lint，减少噪音
allow-needless-raw-string-hashes = true
allow-unreadable-literal = true
allow-doc-markdown = true
```

或在 `src/lib.rs` 顶部添加:

```rust
#![allow(
    clippy::needless_raw_string_hashes,
    clippy::unreadable_literal,
    clippy::doc_markdown,
)]
```
